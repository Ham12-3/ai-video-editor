# ClipForge Test Prompts

A collection of editing prompts to test the full pipeline. Organized by complexity and use case.

## All Features (activates everything)

```
Remove all silences and filler words (um, uh, like, you know), add karaoke-style captions with large white text and dark background at bottom-center, speed up the intro by 1.25x, reframe to 9:16 vertical for TikTok, and add illustrations to visualize the key concepts being discussed
```

## Quick Cleanup

```
Remove all silences longer than 0.5 seconds
```

```
Remove filler words like um, uh, like, you know, basically, and actually
```

```
Trim the first 3 seconds and last 5 seconds of dead air
```

```
Remove silences and filler words, keep 0.1 seconds of padding between cuts so it sounds natural
```

## Speed and Pacing

```

```

```

```

```

```

## Platform Optimization

### TikTok (max 60s)

```
Full TikTok optimization: remove all silences and filler words, add large karaoke captions at bottom center with white text and black background, reframe to 9:16 vertical, trim to 60 seconds max, speed up slow sections by 1.3x
```

### Instagram Reels (max 90s)

```
Optimize for Instagram Reels: reframe to 9:16, add sentence-style captions with medium white text at the bottom, remove dead air and filler words, keep it under 90 seconds, add crossfade transitions between major topic changes
```

### YouTube Shorts (max 60s)

```
YouTube Shorts edit: keep the most engaging 60 seconds, remove all filler words and awkward pauses, add word-by-word captions in large bold text at center, reframe to 9:16
```

### LinkedIn / Professional

```
Clean up for LinkedIn: remove filler words and long pauses, add clean sentence captions at the bottom in small white text, keep the professional tone, do not speed up
```
